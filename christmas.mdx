# 🎄 ChristmasBot Translation Template

Copy this template when creating a new language file.

---

## 📁 File Location

```
/locales/christmasbot/<language-code>.json
```

Example:
```
/locales/christmasbot/it-IT.json
```

---

## 🧩 Template

```json
{
  "commands": {
    "ping": {
      "title": "🏓 Pong! Latency: **{latency} ms**"
    }
  },
  "countdown": {
    "christmas_day": "🎄 Merry Christmas! 🎄",
    "christmas_eve": "🎅 Christmas Eve is here!"
  }
}
```

---

## ⚠️ Rules

- ❌ Do NOT change keys  
- ❌ Do NOT remove `{variables}`  
- ✅ Only translate the text  
- ✅ Keep emojis 🎄🎅  
- ✅ Keep formatting like `**bold**`

---

<Tip>
If you're unsure about a translation, you can leave it in English — we’ll review it!
</Tip>
